# Population Data

This directory contains demographic datasets used by the Wildfire Decision Support System.

## Included Dataset

### indicadores_personas_salta_2022.csv

Population indicators derived from the 2022 census for census tracts ("radios censales") in Salta Province, Argentina.

The dataset is used to estimate potential human exposure and vulnerability in areas affected by wildfire risk.

## Purpose

Population data is incorporated into the risk assessment workflow to:

* Identify potentially affected populations
* Prioritize high-impact risk areas
* Support decision-making and resource allocation
* Complement environmental and meteorological indicators

## Notes

This dataset represents a lightweight demographic layer used for vulnerability analysis within the wildfire monitoring framework.
