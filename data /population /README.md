# Population Data

This directory contains population-related datasets used by the Wildfire Decision Support System.

The project incorporates demographic indicators to estimate potential human exposure and vulnerability in wildfire-prone areas. These variables are integrated with environmental, meteorological and geospatial data to support wildfire risk assessment and decision-making.

## Included Data

* Population indicators for Salta Province (Argentina)
* Census-based demographic information
* Spatially referenced population statistics

## Purpose

Population data is used to:

* Identify areas with higher potential human impact
* Prioritize risk assessment and alert generation
* Support decision-making in wildfire preparedness and response

## Notes

Large datasets and proprietary data sources are not included in this repository. Only sample or publicly shareable resources necessary to understand the project structure are provided.

For full reproducibility, users may need to obtain original datasets from their respective providers.
