# Sample Data

This directory contains small sample datasets used to demonstrate the structure and functionality of the Wildfire Decision Support System.

The samples are intended for inspection, testing and educational purposes.

## Included Data

* Example wildfire observations
* Sample environmental indicators
* Demonstration datasets for model inputs
* Example outputs and intermediate results

## Purpose

Sample datasets allow users to:

* Understand the expected data structure
* Explore the workflow without large downloads
* Test data processing pipelines
* Reproduce selected examples from the project

## Notes

The complete datasets used during development are not included in this repository because of their size and external licensing considerations.

Users interested in reproducing the full workflow should obtain the original data sources referenced in the project documentation.
